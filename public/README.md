
# ArbiProSeller

## Installation Instructions

### Standard Installation
1. Extract all files from the downloaded ZIP file
2. Navigate to the Debug folder
3. Run ArbiProSeller.exe to launch the application

### C: Drive Installation (Recommended)
1. Download both the ZIP file and the installation helper script
2. Place both files in the same folder
3. Right-click on the installation helper script and select "Run as administrator"
4. The application will be installed to C:\ArbiProSeller
5. Use the desktop shortcut created during installation

## System Requirements

### Windows
- Windows 10 or Windows 11
- .NET Runtime (included in Windows 10/11)
- 4GB RAM minimum
- 100MB free disk space

## What's Included in this Package

- ArbiProSeller.exe (Main application)
- Supporting DLL files
- Configuration files
- Documentation
